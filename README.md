## 🔹 Présentation du Projet

**LogiFête PGI** est un Progiciel de Gestion Intégré (PGI/ERP) Web. Il a été conçu pour centraliser l'information dans une base de données unique pour les clients, le stock et les commandes.

* **Problématique** : L'entreprise utilisait des tableurs dispersés, causant des erreurs de stock et des oublis de facturation.
* **Objectifs** : Optimiser la logistique via QR Codes, fiabiliser la facturation et digitaliser les processus via signature électronique.
* **Fonctionnalités Clés** : 
    * **Gestion Commerciale** : Devis, commandes et factures PDF.
    * **Logistique & Stock** : Inventaire en temps réel et gestion des kits.
    * **Maintenance** : Suivi des pannes et calcul automatisé du ROI par produit.
    * **Sécurité** : Gestion des rôles (RBAC), hachage des mots de passe et requêtes préparées PDO.

###

## 🔹 Installation & Déploiement

1.  **Base de données** : Créer une base `logifete` et importer le fichier SQL fourni.
2.  **Configuration** : Modifier les accès `$host`, `$db`, `$user` et `$pass` dans le fichier `db.php`.
3.  **Serveur** : Déployer sur un serveur Linux (Debian/Ubuntu) avec PHP 8.2 minimum.
4.  **Connexion** : Utiliser le compte admin par défaut (boss@logifete.com / logifete2025).
